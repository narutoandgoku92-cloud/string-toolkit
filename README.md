🔤 String Toolkit

🚀 Overview

This is a Python command-line application that performs useful string operations like reversing text, counting vowels, and checking palindromes.

✨ Features

- Reverse text
- Count vowels
- Replace vowels with #
- Check if a string is a palindrome

🛠 Tech Stack

- Python

▶️ How to Run

1. Download or clone the project
2. Open terminal
3. Run:
   python main.py
